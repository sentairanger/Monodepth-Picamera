About Monodepth Estimation UI

Web Interface that uses the Pi Camera Module and the OpenVINO toolkit

This project uses the Pi Camera Module to take images and then performs monodepth estimation using the OpenVINO toolkit. More information is shown here

Capture images

Take images with a single button and that image will be shown in the data portion of the project.

Run Monodepth Estimation

After taking images run monodepth estimation on the image and then display the results in the gallery.

Components of the Project

PiCamera2

  • PiCamera2 GitHub Repository
  • PiCamera2 Manual

This project uses PiCamera2 to control the Camera Module.

OpenVINO

This project utilizes the OpenVINO toolkit to perform monodepth estimation on images.
